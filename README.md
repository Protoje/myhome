# (linux) My Home

Requires:

ctmux
 * zsh
 * vim
 * ssh-agent


Symlink after pull/clone:

 * ln -s ~/git/home/.gitconfig ~/.gitconfig
 * ln -s ~/git/home/.vimrc ~/.vimrc
 * ln -s ~/git/home/.vim ~/.vim
 * ln -s ~/git/home/.zsh/
 * ln -s ~/git/home/.zshenv ~/.zshenv
 * ln -s ~/git/home/.zshrc ~/.zshrc
 * ln -s ~/git/home/.gitmodules ~/.gitmodules
